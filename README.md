# retail-ab-test-analysis
# Анализ эффективности Push-уведомлений (A/B тест)

Проект посвящен анализу маркетингового эксперимента в ритейле. Мы сравниваем эффективность новых Push-уведомлений против классических баннеров.

## 🎯 Задачи проекта
* **Очистка и подготовка данных:** объединение таблиц по транзакциям, клиентам и регионам.
* **Анализ A/B теста:** расчет конверсии в покупку и среднего чека.
* **Сегментация:** проверка результатов в разрезе городов и торговых точек.
* **Автоматизация:** создание Excel-калькулятора для быстрого получения отчетов.

## 🛠 Стек технологий
* **Python** (Pandas, NumPy) — обработка данных.
* **SciPy** — статистическая проверка гипотез.
* **Matplotlib, Seaborn** — визуализация результатов.
* **XlsxWriter** — генерация автоматизированного Excel-отчета.

## 📁 Структура репозитория
* `data/` — исходные данные для анализа.
* `retail-ab-test-analysis.ipynb` — основной процесс исследования (EDA, стат-тесты).
* `ab_test_calculator_final.xlsx` — пример итогового автоматизированного отчета.

## 📈 Краткий вывод Анализа:
*Внедрение пуш-уведомлений привело к статистически значимому росту конверсии и среднего чека во всех регионах (p < 0.05). 
Рекомендуется масштабировать решение на всю сеть и использовать пуши как основной канал коммуникации.*

### 📊 Автоматизированный Excel-инструмент
В рамках проекта разработан динамический калькулятор, позволяющий бизнесу моделировать финансовые результаты A/B теста при изменении параметров:

*   **Настройка параметров:** Возможность изменять размер выборки ($N$) и минимально детектируемый эффект ($MDE$) для оценки потенциальной прибыли.
*   **Финансовые метрики:** Автоматический расчет прироста/падения выручки и общего баланса по успешным и неуспешным торговым точкам.
*   **Прогнозирование для нейтральных сегментов:** Внедрена формула расчета необходимого объема выборки ($n = 16\sigma^2 / MDE^2$) для достижения статзначимости в неопределенных кейсах.
*   **Кастомное форматирование:** Использование библиотеки `xlsxwriter` для создания понятных бизнес-отчетов с автоматическим выделением ключевых показателей.
